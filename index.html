Projects

  • RTP

    Led a research project aimed at redefining tensor parallelism to address memory overhead challenges in neural network training, focuses on memory deduplication and optimal memory allocation.

    demonstrating RTP's capability to achieve performance comparable to established methodologies of DDP and FSDP while supporting larger models with near-linear memory scalability.

  • SuperScalar

    Innovated a new scheduling algorithm for multi-talent distributed training, optimizing system efficiency and performance. Successfully co-designed computation and communication overlapping behavior to reduce system latency.

  • Superbench

    Spearheaded the development of a simulation project for distributed training solutions on large-scale clusters, utilizing high-bandwidth InfiniBand and high-performance NVIDIA A100 GPU. Crafted a user-friendly benchmark tool for non-intrusive, fine-grained monitoring of AI infrastructure, enhancing system transparency and maintainability.

  • LEOpard

    We are building a globally spanning testbed, LEOpard, to quantify these performance opportunities and bottlenecks and their impact on end-to-end transport and applications. Our initial implementation offers unique features to account for LEO dynamics, growing presence across countries, and insights that call for a broad LEO research agenda. We use LEOpard to quantify Starlink performance across countries, demonstrate the opportunity to build LEO network performance predictors, and show their utility in improving application performance.
